### How to Use

Simply open `music-player.html` in your web browser:

```bash
# Direct file open
open music-player.html

# Or serve with a local server
python -m http.server 8000
# Then navigate to http://localhost:8000/music-player.html
```

### Interactive Elements

- **Play/Pause Button** - Click to toggle playback (triggers vinyl animation)
- **Shuffle/Repeat** - Toggle buttons for shuffle and repeat modes
- **Previous/Next** - Navigation buttons (visual feedback only)
- **Progress Bar** - Hover to see progress handle
- **Volume Slider** - Drag to adjust volume (visual feedback)
- **Playlist Toggle** - Click the list icon to expand/collapse playlist

### CSS Techniques Used

- `:checked` pseudo-class for state management
- CSS animations and keyframes
- Gradient backgrounds with animation
- Glassmorphism (backdrop-filter, transparency)
- CSS transitions for smooth interactions
- Pseudo-elements (::before, ::after) for decorative elements
- Advanced selectors for sibling element styling

### Files

- `music-player.html` - HTML structure
- `music-player.css` - All styling and animations

## Getting Started

Simply open `index.html` in your web browser, or serve it using a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using VS Code Live Server extension
# Right-click index.html and select "Open with Live Server"
```

Then navigate to `http://localhost:8000` in your browser.
